Assets Placeholder
==================

Add the following image files to this directory before building:

- icon.png          (1024x1024px, app icon)
- splash.png        (1284x2778px recommended, splash screen image)
- adaptive-icon.png (1024x1024px, Android adaptive icon foreground)
- favicon.png       (48x48px, web favicon)

You can generate placeholder icons quickly using:
  npx expo-image-utils

Or use any image editing tool (Figma, Canva, etc.) to create branded assets.
The app's color scheme is warm cream (#FFF9F0) with orange accents (#FF6B35).
